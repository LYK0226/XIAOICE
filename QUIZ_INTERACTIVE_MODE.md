# 📝 互動式測驗模式

## 🎯 功能說明

新的測驗模式採用**一問一答互動式設計**，類似真實考試體驗：

### ✨ 特色

1. **逐題顯示** - 一次只顯示一題，不會被所有題目淹沒
2. **即時反饋** - 選擇答案後立即顯示正確/錯誤
3. **視覺回饋** - 
   - ✅ **正確答案**：綠色漸層
   - ❌ **錯誤答案**：紅色漸層，同時高亮正確選項
4. **自動前進** - 2秒後自動顯示下一題
5. **完整報告** - 所有題目完成後顯示總結

---

## 🎮 使用流程

### 1️⃣ 開始測驗
點擊紫色「生成測驗」按鈕 📋

### 2️⃣ 測驗開始提示
```
📋 PDF 測驗開始
共 5 題選擇題
Test ID: xxx-xxx-xxx
```

### 3️⃣ 逐題作答
```
❓ 第 1/5 題

SSC103 Introduction to Psychology

1. SSC103 Introduction to Psychology
2. 文檔中沒有相關描述
3. •Lecturer 3 – Developmental Psychology•Dr. Stephen Chan
4. 以上皆非
```

- 點擊任一選項
- 立即顯示結果：
  - ✅ 正確 → 綠色高亮
  - ❌ 錯誤 → 紅色高亮 + 顯示正確答案（綠色）

### 4️⃣ 自動下一題
等待 2 秒自動進入下一題

### 5️⃣ 完成測驗
```
🏆 測驗完成！

80.0%

答對 4 題 / 共 5 題

📊 詳細結果：
Q1: ✅ 正確
   SSC103 Introduction to Psychology

Q2: ❌ 錯誤
   What is Developmental Psychology?
   您的答案: 以上皆非
   正確答案: What is Developmental Psychology?
...
```

## 🎨 視覺設計

### 顏色方案

| 元素 | 顏色 |
|------|------|
| 測驗啟動 | 紫色漸層 (#667eea → #764ba2) |
| 題目卡片 | 白底 + 紫色邊框 |
| 正確答案 | 綠色漸層 (#11998e → #38ef7d) |
| 錯誤答案 | 紅色漸層 (#eb3349 → #f45c43) |
| 結果報告 | 綠色漸層 (#11998e → #38ef7d) |

### 互動效果

- **懸停效果**：淺藍色背景 (#e8f0fe)
- **選中效果**：漸層背景 + 白色文字 + 圖標
- **禁用效果**：點擊後所有按鈕禁用，防止重複點擊

## 🔧 技術實現

### 主要函數

```javascript
// 1. 開始互動式測驗
startInteractiveQuiz(quizData)
  - 顯示測驗開始訊息
  - 初始化測驗狀態
  - 啟動第一題

// 2. 顯示下一題
showNextQuestion(quizState)
  - 渲染題目和選項
  - 處理答案選擇
  - 顯示即時反饋
  - 自動進入下一題

// 3. 顯示最終結果
showQuizResults(quizState)
  - 計算分數
  - 顯示詳細報告
  - 提交到後端記錄
```

### 測驗狀態

```javascript
{
    test_id: "uuid",
    questions: [...],
    currentIndex: 0,
    userAnswers: [],
    correctCount: 0
}
```

## 📊 與舊版本對比

| 特性 | 舊版本 | 新版本 |
|------|--------|--------|
| 顯示方式 | 一次顯示全部5題 | 逐題顯示 |
| 反饋時機 | 提交後才知道結果 | 選擇後立即反饋 |
| 用戶體驗 | 需要記住答案 | 即時知道對錯 |
| 視覺負擔 | 頁面內容多 | 專注當前題目 |
| 互動性 | 低 | 高 |
| 答題節奏 | 自主控制 | 自動前進（2秒） |

---

## 🚀 優勢

1. **降低認知負荷** - 一次只處理一個問題
2. **即時學習** - 錯誤答案立即看到正確解答
3. **更好的參與度** - 互動式體驗更有趣
4. **避免跳題** - 強制按順序完成
5. **視覺引導** - 清晰的進度指示（第 X/Y 題）
