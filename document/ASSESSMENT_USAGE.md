2. **在主页sidebar添加"兒童評估"按钮**

4. **配置视频示范功能**

## 📝 如何使用评估系统

### 方法1：从主页进入

1. 登录系统后，在左侧侧边栏找到 **"兒童評估"** 按钮
2. 点击按钮进入评估页面

### 方法2：直接访问

访问：`http://127.0.0.1:5000/child_assessment`

## 🚀 开始评估

### 步骤1：填写基本信息
1. 点击 **"Upload Video / YouTube Link"** 按钮
2. 填写以下信息：
   - **Assessment Name**: 评估名称（如：2025年12月评估）
   - **Child Name**: 儿童姓名
   - **Child Age**: 儿童年龄（月数，0-84个月）
   - **Video/YouTube**: 可选，暂时可以留空

### 步骤2：开始评估
1. 点击 **"Start"** 按钮开始评估
2. 系统会显示第一题

### 步骤3：观看视频并评估
对于每一题：
1. **观看示范视频** - 视频会自动循环播放
2. **让孩子模仿动作**
3. **选择结果**：
   - 点击 **绿色"做到"按钮** ✓ 如果孩子能完成动作
   - 点击 **红色"做不到"按钮** ✗ 如果孩子不能完成

### 步骤4：完成所有题目
- 系统会自动进入下一题
- 进度条显示当前进度
- 共有5道题目

### 步骤5：查看结果
完成所有题目后：
- 查看 **DQ分数**（发育商）
- 查看 **评估等级**
- 查看 **详细报告**

## 📊 评估内容

当前包含5道评估题：

1. **🐻 举起双手** - 大运动能力
2. **🐻 拍手** - 精细动作能力
3. **🐻 踢腿** - 大运动能力
4. **🐻 挥手（再见）** - 精细动作能力
5. **🐻 蹲下** - 大运动能力

## 🎬 关于示范视频

当前使用的是演示视频。要替换成真实的熊示范视频：

1. 准备5个MP4格式的视频文件
2. 放到 `/app/static/upload/videos/` 目录
3. 编辑 `/app/static/js/child_assessment.js`
4. 更新每题的 `videoUrl` 字段

详细说明请参考：**VIDEO_SETUP_GUIDE.md**

## 🔍 DQ评分标准

DQ (发育商) = (发育年龄 / 实际年龄) × 100

**等级分类**：
- **DQ ≥ 130**: 优异
- **DQ 116-129**: 中上
- **DQ 85-115**: 正常
- **DQ 71-84**: 边界低
- **DQ < 71**: 需要关注

## 💡 提示

- 视频可以**重复播放**，让孩子多看几次
- 评估时保持**轻松愉快**的氛围
- 如果孩子不配合，可以**暂停**后再继续
- 点击 **"返回主菜单"** 可以随时退出

## 🐛 故障排除

### 问题：点击按钮没反应
**解决**：检查浏览器控制台是否有JavaScript错误

### 问题：视频无法播放
**解决**：
1. 检查网络连接
2. 尝试刷新页面
3. 更换浏览器

### 问题：评估未保存
**解决**：确保完成所有题目并点击"提交评估"
