# Product Overview

XIAOICE is an intelligent chat assistant with multimodal AI capabilities and real-time pose detection.

## Core Features

- **Multi-agent AI System**: Uses Google ADK with specialized agents (coordinator, text, media) for intelligent task routing
- **Real-time Chat**: WebSocket-based streaming responses with conversation history
- **Multimodal Support**: Analyzes text, images, and videos (up to 500MB)
- **Pose Detection**: Real-time human pose tracking and action recognition via webcam using MediaPipe
- **Secure Authentication**: JWT-based auth with encrypted API key storage
- **Multi-language**: Supports Traditional Chinese, English, and Japanese
- **Customizable**: User preferences for themes, language, and AI models

## Target Users

Users who need an AI assistant that can handle both text conversations and visual content analysis, with optional pose detection capabilities for fitness, accessibility, or interactive applications.

## Key Differentiators

- Multi-agent architecture for specialized task handling
- Real-time pose detection with action recognition
- Secure per-user API key management with encryption
- Persistent conversation sessions tied to database
