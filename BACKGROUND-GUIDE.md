# 🎨 背景自定義指南 / Background Customization Guide

## 功能概述 / Overview

现在您可以完全自定义聊天应用的背景！支持三种背景类型：

1. **渐变色 (Gradient)** - 8 种预设 + 自定义双色渐变
2. **纯色 (Solid Color)** - 10 种预设 + 自定义颜色
3. **图片 (Image)** - 上传您喜欢的背景图片

---

## 📝 使用方法 / How to Use

### 第一步：打开设置
1. 点击左侧边栏的 **"设置"** 按钮（齿轮图标）
2. 在弹出的窗口中，点击 **"背景 / Background"** 标签页

### 第二步：选择背景类型

#### 🌈 选项 1：渐变色 (Gradient)

**使用预设渐变：**
1. 点击 **"渐变色 / Gradient"** 按钮
2. 从 8 种精美预设中选择一个：
   - 紫色梦幻（默认）
   - 粉红激情
   - 清新蓝天
   - 薄荷绿意
   - 日落黄昏
   - 神秘深海
   - 柔和梦境
   - 甜蜜浪漫
3. 点击即可应用

**创建自定义渐变：**
1. 在 "自定义渐变" 区域
2. 选择颜色 1（左侧颜色）
3. 选择颜色 2（右侧颜色）
4. 点击 **"应用自定义渐变"** 按钮

#### 🎨 选项 2：纯色 (Solid Color)

**使用预设颜色：**
1. 点击 **"纯色 / Solid"** 按钮
2. 从 10 种预设颜色中选择
3. 点击色块即可应用

**使用自定义颜色：**
1. 点击颜色选择器
2. 选择您喜欢的颜色
3. 点击 **"应用"** 按钮

#### 🖼️ 选项 3：背景图片 (Image)

**上传图片：**
1. 点击 **"图片 / Image"** 按钮
2. 点击 **"选择图片"** 或直接点击预览区域
3. 从您的电脑选择图片
4. 图片会自动适配屏幕尺寸

**支持格式：**
- JPEG / JPG
- PNG
- WebP
- GIF

**清除图片：**
- 点击 **"清除背景"** 按钮即可移除图片

### 第三步：重置背景
如果想恢复默认背景，点击底部的 **"恢复默认背景 / Reset to Default"** 按钮。

---

## 💾 自动保存 / Auto-Save

所有背景设置都会自动保存到浏览器本地存储，下次打开应用时会自动加载您的设置！

## 🎯 最佳实践 / Best Practices

### 推荐的背景选择

**适合长时间使用：**
- 柔和梦境（浅色渐变）
- 清新蓝天（蓝色调）
- 薄荷绿意（绿色调）

**适合夜间使用：**
- 神秘深海（深色渐变）
- 深灰色或深蓝色纯色背景

**适合专业场景：**
- 纯色背景（深灰、深蓝）
- 简洁的双色渐变

### 图片建议

**推荐尺寸：**
- 1920x1080 或更高分辨率
- 保持 16:9 宽高比

**推荐风格：**
- 简洁的抽象图案
- 柔和的风景照片
- 低对比度的纹理背景
- 避免过于复杂或高对比的图片（影响文字阅读）

**文件大小：**
- 建议小于 5MB
- 大文件可能影响加载速度

## 🔧 技术细节 / Technical Details

### 本地存储键名
- `bgType`: 背景类型（gradient/solid/image）
- `bgValue`: 背景值（CSS 代码或图片 Base64）

### CSS 属性
背景通过 `document.body.style.background` 应用

### 数据保存
- 渐变色和纯色：保存为 CSS 字符串
- 图片：保存为 Base64 编码（存储在 localStorage）

---

## ⚠️ 注意事项 / Notes

1. **图片大小限制**
   - localStorage 有 5-10MB 的限制
   - 过大的图片可能无法保存

2. **浏览器兼容性**
   - 现代浏览器完全支持
   - IE 浏览器可能有兼容问题

3. **隐私模式**
   - 隐私/无痕模式下设置不会保存

4. **清除数据**
   - 清除浏览器缓存会删除背景设置
