## 自定义技巧 / Customization Tips

### 创建和谐的渐变

**同色系渐变**
- 选择同一色系的深浅两色
- 例如：浅蓝 → 深蓝

**对比色渐变**
- 选择色轮上相对的颜色
- 例如：橙色 → 蓝色

**相邻色渐变**
- 选择色轮上相邻的颜色
- 例如：紫色 → 粉色

### 选择合适的背景图片

**推荐图片类型**
1. 抽象几何图案
2. 柔和的纹理（如水彩、渐变）
3. 简洁的风景照（低饱和度）
4. 简约的几何图形

**避免使用**
1. 高对比度图片
2. 过于复杂的图案
3. 文字密集的图片
4. 太暗或太亮的图片

**图片处理建议**
- 降低饱和度到 50-70%
- 增加轻微模糊效果
- 调整透明度到 80-90%
- 使用 overlay 效果

## 常见问题 / FAQ

**Q: 背景设置会影响性能吗？**
A: 渐变和纯色几乎无影响；图片背景可能略微增加内存占用。

**Q: 可以设置动态背景吗？**
A: 当前版本不支持，未来可能添加此功能。

**Q: 背景图片会上传到服务器吗？**
A: 不会，所有数据都保存在本地浏览器中。

**Q: 如何分享我的背景设置？**
A: 可以复制 CSS 代码或导出配置文件（即将支持）。
