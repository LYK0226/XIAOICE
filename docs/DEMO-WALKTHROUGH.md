# 🎯 功能演示说明

## 页面导航功能展示

### 功能 1: 从聊天页面跳转到功能展示

**操作步骤**:
1. 打开聊天应用 (`http://localhost:8000/app/templates/index.html`)
2. 在顶部语言栏右侧找到"功能展示"按钮
3. 点击按钮
4. 自动跳转到功能展示页面

**位置**: 顶部语言栏，三个语言按钮右侧  
**按钮外观**: 半透明白色背景，带有信息图标  
**悬停效果**: 背景加深，按钮上浮

---

### 功能 2: 从功能展示页面返回聊天

**方式 A - 右上角浮动按钮**:
1. 在功能展示页面 (`http://localhost:8000/app/templates/demo.html`)
2. 查看右上角固定位置的"返回聊天"按钮
3. 点击按钮
4. 立即返回聊天界面

**位置**: 右上角固定定位  
**按钮外观**: 白色背景，蓝紫色文字  
**悬停效果**: 渐变紫色背景，白色文字

**方式 B - 主按钮**:
1. 在功能展示页面向下滚动
2. 找到中部的"🚀 立即开始使用"大按钮
3. 点击按钮
4. 跳转到聊天界面开始使用

**位置**: 页面中部，功能展示后  
**按钮外观**: 大号白色圆角按钮  
**悬停效果**: 放大效果

## 常见问题

**Q: 按钮可以自定义颜色吗？**  
A: 可以，编辑 styles.css 中的 `.page-switch-btn` 和 `.back-to-app` 类。

**Q: 可以添加更多页面吗？**  
A: 可以，只需创建新页面并添加相应的导航按钮。

**Q: 为什么使用固定定位？**  
A: 演示页面较长，固定定位保证按钮始终可见，方便用户随时返回。

**Q: 可以改变按钮位置吗？**  
A: 可以，修改 CSS 中的 `position`、`top`、`right` 等属性。

**Q: 支持键盘导航吗？**  
A: 支持，可以使用 Tab 键选择，Enter 键激活。
