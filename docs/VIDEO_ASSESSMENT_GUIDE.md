# 視頻示範評估功能使用指南

## 功能概述

新的視頻示範評估功能讓家長能夠通過觀看示範視頻來評估兒童的發育狀況。每道評估題目都會顯示一個示範視頻（例如熊舉手），然後家長根據孩子的實際情況選擇"做到"或"做不到"。

## 功能特點

✅ **視頻示範**: 每題配有清晰的動作示範視頻  
✅ **簡單評估**: 只需點擊"做到"或"做不到"按鈕  
✅ **進度追蹤**: 實時顯示評估進度條  
✅ **即時結果**: 完成後立即查看評估結果和DQ分數  

## 使用流程

### 1. 開始評估
1. 登入系統後進入**Assessments**頁面
2. 點擊**Upload Video / YouTube Link**按鈕
3. 填寫評估信息：
   - Assessment Name（評估名稱）
   - Child Name（兒童姓名）
   - Child Age（兒童年齡，以月為單位）
4. 點擊**Start**開始評估

### 2. 進行評估
每道題目包含：
- 🎬 **示範視頻**: 自動播放，可重複觀看
- 📋 **評估說明**: 詳細描述需要觀察的動作
- ✅ **做到按鈕**: 綠色按鈕，孩子能完成該動作時點擊
- ❌ **做不到按鈕**: 紅色按鈕，孩子無法完成該動作時點擊

### 3. 評估題目示例
共5道題目，涵蓋不同發育領域：

1. **大運動 - 舉起雙手** 🐻  
   觀察兒童是否能將雙手舉起到頭部上方

2. **精細動作 - 拍手** 🐻  
   觀察兒童是否能雙手合掌拍打

3. **大運動 - 踢腿** 🐻  
   觀察兒童是否能抬起一隻腿做踢腿動作

4. **精細動作 - 揮手（再見）** 🐻  
   觀察兒童是否能做出揮手告別的動作

5. **大運動 - 蹲下** 🐻  
   觀察兒童是否能從站立姿勢蹲下

### 4. 查看結果
完成所有題目後：
- 系統自動計算**發育商 (DQ)**
- 顯示評估詳情和完成率
- 提供發育等級說明
- 可選擇返回主菜單

## DQ評分標準

| DQ範圍 | 等級 | 說明 |
|--------|------|------|
| ≥ 130 | 優異 | 發育水平優秀 |
| 116-129 | 中上 | 發育水平良好 |
| 85-115 | 正常 | 發育水平正常 |
| 71-84 | 邊界低 | 需要關注 |
| < 71 | 障礙 | 建議專業評估 |

## 自定義視頻

如需添加自己的示範視頻：

1. 準備視頻文件（MP4格式推薦）
2. 將視頻放置在 `/app/static/upload/videos/` 目錄
3. 在 `child_assessment.js` 的 `loadQuestions()` 方法中更新 `videoUrl`：

```javascript
{
    id: 1,
    domain: '大運動',
    emoji: '🐻',
    question: '兒童能否舉起雙手？',
    description: '觀察兒童是否能模仿影片中的動作，將雙手舉起到頭部上方。',
    videoUrl: '/static/upload/videos/your-video.mp4'  // 修改此處
}
```

## 注意事項

⚠️ **視頻格式**: 確保視頻為瀏覽器支持的格式（MP4, WebM等）  
⚠️ **視頻大小**: 建議視頻文件不超過50MB以確保流暢播放  
⚠️ **網絡連接**: 使用在線視頻時需要穩定的網絡連接  
⚠️ **隱私保護**: 不要在示範視頻中包含敏感個人信息
