## 🚀 部署和使用

### 快速开始
```bash
# 1. 获取 API 密钥
访问 https://makersuite.google.com/app/apikey

# 2. 配置密钥（自动）
./setup.sh YOUR_API_KEY

# 3. 启动服务器
python3 -m http.server 8000

# 4. 打开浏览器
访问 http://localhost:8000/app/templates/index.html
```

### 手动配置
1. 编辑 `app/static/js/chatbox.js` 第 21 行
2. 替换 API 密钥
3. 保存文件
4. 启动服务器
